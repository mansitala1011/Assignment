What is JQuery?

⇒jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make
it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines
of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.


How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

⇒css:The css() method sets or returns one or more style properties for the selected elements.
class:The addClass() method adds one or more class names to the selected elements.
Remove class:The removeClass() method removes one or more class names from the selected elements.
The jQuery animate() method is used to create custom animations.
